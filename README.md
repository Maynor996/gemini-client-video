# MaynorAPI Pro Promotional Video (Remotion)

This project contains the source code for a 31-second vertical (1080x1920) promotional video for **MaynorAPI Pro** (https://apipro.maynor1024.live).

## 🎬 Video Overview
- **Objective**: Showcase MaynorAPI Pro as the most professional AI API aggregation platform.
- **Key Models**: 300+ models including GPT-4o, Claude 3.5, Gemini, DeepSeek.
- **USP**: 18 months stability, 12ms latency, $1.2/1k tokens.
- **Visual Style**: Tech Blue (#007bff) and Modern White.

## 🚀 How to Run

### Install
```bash
npm install
```

### Preview
```bash
npm start
```

### Local Render
```bash
npm run build
```

## ☁️ Cloud Rendering (GitHub Actions)
Every time you push to the `main` branch, a high-quality video is rendered automatically.
1. Go to the [Actions](https://github.com/Maynor996/maynorapipro-video/actions) tab.
2. Select the latest run.
3. Download the `remotion-video` artifact to get your MP4 file.

## 📂 Project Structure
- `src/SceneIntro.tsx`: Opening animation.
- `src/SceneFeatures.tsx`: AI Model matrix.
- `src/SceneAdvantages.tsx`: Pricing and stability.
- `src/SceneCTA.tsx`: Final call-to-action.
- `public/`: Assets and screenshots.
